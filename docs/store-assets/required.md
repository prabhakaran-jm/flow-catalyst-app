# Required Store Listing Fields

These fields are **mandatory** for both Google Play and App Store. Fill in before submitting.

---

## Privacy Policy URL

**Status:** Required for both stores  
**Format:** Public HTTPS URL

**Template:** `https://your-domain.com/privacy`

**Content:** See [docs/legal/PRIVACY_POLICY.md](../legal/PRIVACY_POLICY.md) for a ready-to-use template covering:
- Data collected (email, usage, subscription status)
- Third-party services (Supabase, RevenueCat, AI providers)
- User rights (access, deletion, export)
- Contact for privacy inquiries

**Before publishing:** Replace `[DATE]` and `[SUPPORT_EMAIL]` placeholders.

**Where to host:** Your website, GitHub Pages, or a service like [Termly](https://termly.io), [PrivacyPolicies.com](https://www.privacypolicies.com).

---

## Support Email

**Status:** Required for both stores  
**Format:** Valid email address

**Template:** `support@your-domain.com`

**Used for:** User contact, store listing, crash reports.

---

## Summary

| Field | Google Play | App Store |
|-------|-------------|-----------|
| Privacy Policy URL | ✓ Required | ✓ Required |
| Support Email | ✓ Required | ✓ Required |
